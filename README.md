Веб-приложение для управления библиотекой, позволяющее добавлять, искать, брать и возвращать книги. Реализовано на Java с использованием Spring Boot и SQL.
Стек технологий:
Backend: Java 21, Spring Boot 3.5.9
Фреймворки: Spring MVC, Spring Data JPA, Spring Validation
База данных: H2 Database для разработки
Сборка: Maven
