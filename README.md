# Library - система управления библиотекой

## REST API для управления библиотекой
- Полный CRUD для книг и полок через RESTful API
- Интуитивно понятные HTTP endpoints
- Быстрый доступ ко всем функциям библиотеки

## Две основные сущности

### Книги
- Создание, чтение, обновление и удаление книг
- Поиск книг по различным критериям
- Сортировка по названию, автору и году издания
- Привязка книг к полкам

### Полки
- Управление полками для организации книг
- Просмотр книг на конкретной полке
- Перемещение книг между полками
- Очистка полок от книг

## Расширенный поиск и фильтрация
- Поиск по ключевому слову в названии и авторе
- Фильтрация по автору и году издания
- Комбинированный поиск по названию и автору
- Пагинация результатов для больших коллекций

## Быстрая и надежная работа
- Построен на основе Spring Boot и Spring Data JPA
- Оптимизированные запросы к базе данных
- Обработка ошибок с понятными сообщениями
- Полное логирование всех операций

# Технологии

## Java 21
## Spring Boot 3.5.9
## Spring Data JPA с PostgreSQL
## Maven
